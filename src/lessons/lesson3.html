Functions in Python

Functions are reusable pieces of code that perform a specific task. They help to organize code and make it more readable.

Defining a Function

def function_name(parameters):
    # code to execute
    return result

Calling a Function

To use a function, you simply call it by its name and pass the required arguments.

result = function_name(arguments)

Modules in Python

Modules are files containing Python code that can define functions, classes, and variables. They allow for code organization and reuse.

Importing a Module

import module_name

Using Functions from a Module

module_name.function_name(arguments)
