Introduction to Python

Welcome to the first lesson on Python programming! In this lesson, we will cover the basics of Python, including:

  • Variables
  • Data Types
  • Basic Syntax

Variables

In Python, a variable is a name that refers to a value. You can create a variable by simply assigning a value to it:

my_variable = 10

Data Types

Python has several built-in data types, including:

  • Integers: Whole numbers, e.g., 1, 2, 3
  • Floats: Decimal numbers, e.g., 1.5, 2.0
  • Strings: Text, e.g., "Hello, World!"
  • Booleans: True or False values

Basic Syntax

Python uses indentation to define blocks of code. Here is an example of a simple if statement:

if my_variable > 5:
    print("Variable is greater than 5")
